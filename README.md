# Angular Portfolio Starter (Standalone + Tailwind)

A clean, modern single-page portfolio built with Angular 18 (standalone components) and TailwindCSS.

## Quick Start
1. Install Node.js 18+
2. In the project folder, run:
   ```bash
   npm install
   npm run start
   ```
   This uses the local Angular CLI from devDependencies.

## Deploy (GitHub Pages, optional)
- Build: `npm run build`
- Deploy the `dist/angular-portfolio-starter` folder to any static host (GitHub Pages, Netlify, Vercel, etc.).

## Customize
- Edit text in the components under `src/app/components/*`
- Update skills, projects, and contact info quickly.
